## ðŸ“ Technical Specifications

### Image Formats
- **Icons**: PNG with transparency
- **Photos**: JPG (optimized, 80-85% quality)
- **Backgrounds**: PNG for patterns, JPG for photos
- **Thumbnails**: JPG (optimized)

### Size Guidelines
- **App Icon**: 1024x1024px
- **Feature Icons**: 512x512px
- **Backgrounds**: 1080x1920px (portrait) or 1920x1080px (landscape)
- **Thumbnails**: 1280x720px (16:9) or 800x800px (square)
- **Event Cards**: 1200x800px

### Optimization
```bash
# Use ImageMagick for optimization
magick input.png -resize 512x512 -quality 85 output.png

# Or use online tools:
# - TinyPNG (https://tinypng.com)
# - Squoosh (https://squoosh.app)
```

## ðŸ“¦ Asset Integration

### Add to pubspec.yaml
```yaml
flutter:
  assets:
    - assets/images/
    - assets/icons/
```

### Use in Code
```dart
// Images
Image.asset('assets/images/home_background.png')

// Icons
Image.asset('assets/icons/meditation.png', width: 48, height: 48)

// With caching for network images
CachedNetworkImage(
  imageUrl: url,
  placeholder: (context, url) => Image.asset('assets/images/placeholder.png'),
)
```
